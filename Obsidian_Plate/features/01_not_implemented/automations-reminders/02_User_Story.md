## Main user story
As a user who wants to build a cooking habit,
I want to receive push notifications reminding me of my meals and shopping days,
So that I don't forget to cook or shop.

## Target users
- Alle aktiven Nutzer die Erinnerungen wollen

## Main flow
1. Nutzer konfiguriert Reminder in Settings
2. Wählt Zeiten für Mahlzeiten-Erinnerungen
3. Wählt Einkaufstag-Erinnerung
4. Erhält Push-Notification zur konfigurierten Zeit

## Alternative flows
- Alle Notifications deaktivieren

## Error cases
- Permission verweigert: Hinweis in Settings

## Expected result
Pünktliche Push-Notifications für Mahlzeiten und Einkaufstage.
