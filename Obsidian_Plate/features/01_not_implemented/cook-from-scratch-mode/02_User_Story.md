## Main user story
As a health-conscious user,
I want to enable a "cook from scratch" mode,
So that my meal plans only include recipes with whole, unprocessed ingredients.

## Target users
- Gesundheitsbewusste Nutzer
- Nutzer die keine Fertigprodukte verwenden wollen

## Main flow
1. Nutzer öffnet Profile/Settings
2. Aktiviert "Cook from Scratch" Toggle
3. Nächster generierter Plan enthält nur Rezepte mit unverarbeiteten Zutaten

## Alternative flows
- Deaktivieren: nächster Plan normal

## Error cases
Keine spezifischen

## Expected result
Plan ohne Fertigprodukte oder stark verarbeitete Zutaten.
