## Feature name
Favorites Management

## Purpose
Nutzer können Rezepte als Favoriten markieren und eine kuratierte Favoriten-Liste aufbauen.

## Problem
`user_favorites` Tabelle existiert bereits, aber es gibt keine dedizierte UI und minimale Integration im Plan-Flow.

## Goal
Nutzer kann Rezepte favorisieren, eine Favoriten-Liste einsehen und Favoriten bevorzugt in Plan-Generierung einbeziehen.

## Business / product value
Erhöht Retention: Nutzer bauen eine persönliche Bibliothek bevorzugter Rezepte auf.

## Scope
- Favoriten-Button auf Recipe Detail Screen
- Favoriten-Tab oder Sektion in Recipes Tab
- Favoriten in Plan-Generierung als "bevorzugt" einfließen lassen

## Out of scope
Öffentliche Favoriten-Listen, Teilen von Favoriten
