## Main user story
As a user,
I want to browse my recipe library and add my own recipes,
So that I have a growing collection of meals I can use in my plan.

## Target users
- Aktive Nutzer die ihre Rezepte verwalten wollen

## Main flow
1. Nutzer öffnet Recipes Tab
2. Sieht AI-generierte Rezepte (History) + eigene (Manual)
3. Tippt auf Rezept → Detail mit Zutaten + Schritten
4. Oder: Tippt "+" → neues Rezept manuell anlegen

## Alternative flows
- Rezepte filtern nach Kategorie / Küche

## Error cases
- Speichern schlägt fehl: Toast-Fehlermeldung

## Expected result
Nutzer hat Zugriff auf alle Rezepte und kann neue anlegen.
