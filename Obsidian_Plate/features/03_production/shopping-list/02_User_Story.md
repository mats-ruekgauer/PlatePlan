## Main user story
As a user with a weekly meal plan,
I want to generate a complete shopping list automatically,
So that I don't have to manually figure out what to buy.

## Target users
- Alle aktiven Nutzer (wöchentlich)

## Main flow
1. Nutzer öffnet Shopping Tab
2. Tippt "Liste generieren"
3. Liste erscheint kategorisiert (Gemüse, Fleisch, Milchprodukte, etc.)
4. Nutzer hakt Artikel beim Einkaufen ab

## Alternative flows
- Liste manuell ergänzen

## Error cases
- Kein Plan vorhanden: CTA "Erst Plan generieren"

## Expected result
Vollständige, kategorisierte Einkaufsliste für die aktuelle Woche.
