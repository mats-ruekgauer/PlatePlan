## Main user story
As a user living with others,
I want to create a shared household and invite my family/flatmates,
So that we all have the same meal plan that considers everyone's preferences.

## Target users
- Familien, WGs, Paare

## Main flow
1. Nutzer erstellt Household in der App
2. Erhält Invite-Link
3. Teilt Link mit Mitbewohnern
4. Mitbewohner tritt via Link bei
5. Nächster generierter Plan berücksichtigt alle Mitglieder-Präferenzen

## Alternative flows
- Invite-Link rotieren wenn er abgelaufen/kompromittiert

## Error cases
- Ungültiger Token: Fehlermeldung
- Bereits Mitglied: entsprechende Meldung

## Expected result
Gemeinsamer Haushalt mit geteiltem Meal Plan.
