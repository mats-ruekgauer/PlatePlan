## Main user story
As a user with a generated meal plan,
I want to see my weekly meals clearly organized by day and meal slot,
So that I know what to cook each day.

## Target users
- Alle aktiven Nutzer (tägliche Nutzung)

## Main flow
1. Nutzer öffnet Home Tab → sieht aktuellen Wochenplan
2. Scrollt durch Tage / wechselt Woche
3. Tippt auf Mahlzeit → Meal Detail Screen
4. Markiert Mahlzeit als "gekocht"

## Alternative flows
- Woche navigieren: vor/zurück Buttons
- Meal ersetzen: aus Meal Detail heraus

## Error cases
- Kein Plan vorhanden: CTA "Plan generieren" anzeigen

## Expected result
Nutzer sieht alle Mahlzeiten der Woche und kann sie verwalten.
