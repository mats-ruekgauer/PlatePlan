| Token | Hex | Verwendung |
|-------|-----|------------|
| `primary` | `#2D6A4F` | Buttons, aktive States, CTAs |
| `primaryLight` | `#52B788` | Hover, sekundäre Highlights |
| `primaryXLight` | `#D8F3DC` | Backgrounds, Chips, Tags |
| `surface` | `#FFFFFF` | Card-Hintergründe, Modals |
| `surfaceSecondary` | `#F8F9FA` | Screen-Hintergründe |
| `textPrimary` | `#1A1A2E` | Haupttext |
| `textSecondary` | `#6B7280` | Untertitel, Labels |
| `textMuted` | `#9CA3AF` | Platzhalter, Disabled |
| `border` | `#E5E7EB` | Divider, Input-Rahmen |
| `danger` | `#DC2626` | Fehler, destructive Actions |
| `warning` | `#D97706` | Warnungen |
| `success` | `#16A34A` | Bestätigungen, grüne States |
| `calorie` | `#F59E0B` | Kalorien-Anzeige |
